allowed-tools: Bash
description: Remove a worktree
---

If $ARGUMENTS is empty or contains spaces, stop here and ask the user to provide a worktree name in dash-case format.
If $ARGUMENTS is provided, verify it is in dash-case format and remove the worktree named $ARGUMENTS.

## Output

Double check and simply confirm the worktree $ARGUMENTS as been removed and the new worktree name you are now in.
